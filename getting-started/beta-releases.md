# Beta Releases

We generally will release a beta version of Chromatics for public testing and feedback before we submit it for general release. These beta releases can be buggy by their nature, but will also have the latest additions, bug fixes and performance improvements.

To access beta releases, please join our Discord server and check out the channel #beta-releases. We are always wanting feedback on how Chromatics is working and how we can improve.
