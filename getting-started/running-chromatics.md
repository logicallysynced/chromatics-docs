# Running Chromatics

When running Chromatics, the recommended order of opening applications is:

1. Final Fantasy XIV (Game running).
2. Chromatics
3. Any other third-party apps you wish to run.



## Closing Chromatics

By default, clicking the red X button in Chromatics will only minimize Chromatics to the taskbar and not fully quit the application. To close Chromatics, **find the Chromatics logo in the taskbar** (the menu just to the left of the system clock), **right-click** and select **Close**.
